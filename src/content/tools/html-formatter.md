### A Deep Dive into the Evolution of Web Markup

Before the "Prettier Era," developers formatted code by hand or relied on simplistic browser extensions that often broke sensitive spacing in `pre` tags or scripts. 

- **The Table Era:** In the early web, HTML was a mess of nested tables used for layout, making readability nearly impossible without strict formatting rules.
- **The Semantic Era:** The arrival of HTML5 introduced meaningful tags like `<article>` and `<section>`, making vertical indentation even more critical for identifying the "Logic of the Page."
