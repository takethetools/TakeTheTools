## About the ZIP Code Extractor

The **ZIP Code Extractor** scans unstructured text and automatically pulls out all US ZIP codes and international postal codes. Perfect for data processing, marketing segmentation, and geographic analysis.

### What ZIP Codes Are Detected

**US ZIP Codes:**
- Standard 5-digit: `10001`
- ZIP+4 extended: `10001-1234`

**International Postal Codes:**
- UK: `SW1A 1AA`
- Canada: `M5H 2N2`
- Australia: `2000`
- Germany: `10115`

### Use Cases

**Direct Mail Campaigns:** Extract ZIP codes from customer databases or address blocks to segment mailing lists by region.

**Geographic Analytics:** Analyze where your customers, leads, or contacts are located by extracting ZIP codes from raw address data.

**E-commerce Shipping:** Validate or parse shipping addresses by extracting the postal code from unstructured address text.

**Real Estate:** Extract property ZIP codes from listings, documents, or spreadsheet data for geographic market analysis.

### How It Works

The tool uses pattern matching (regular expressions) to detect sequences that match known postal code formats. It highlights each match and presents a clean, deduplicated list for easy copying or export.
