## About the Markdown to PDF Converter

The **Markdown to PDF Converter** transforms `.md` Markdown files and text into beautifully formatted PDF documents, ready for sharing, printing, or professional distribution.

### Why Convert Markdown to PDF?

**Professional Documents**
Markdown is great for writing, but PDFs are the standard for sharing professional documents — reports, proposals, documentation, and resumes. This tool bridges that gap.

**Documentation Distribution**
Technical documentation written in Markdown (for GitHub or internal wikis) often needs to be shared as PDFs with non-technical stakeholders.

**README to Report**
Convert GitHub README files or project documentation into standalone PDF reports.

**Printable Content**
Markdown content can't be printed directly with nice formatting. Converting to PDF first ensures proper page layout and typography.

### Markdown Elements Supported

- **Headings** (H1–H6) with proper typography hierarchy
- **Bold**, *italic*, and `code` inline formatting
- Ordered and unordered lists
- Code blocks with syntax highlighting
- Tables
- Blockquotes
- Horizontal rules
- Links and images

### PDF Features

- Clean, professional typography
- Proper page margins and padding
- Page numbers
- A4 and Letter size options
- Preserves code block formatting with monospace font

### Tips for Best Results

- Use heading levels consistently (H1 for title, H2 for sections)
- Keep tables simple — very wide tables may overflow page width
- Large images will be scaled to fit the page
