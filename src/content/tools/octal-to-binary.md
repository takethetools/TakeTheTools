### Frequently Asked Questions (FAQs)

1. **What is Octal?** A base-8 number system that uses digits 0 through 7.
2. **Why do computers use base-8?** Historically, it was easier to group 3 bits at a time on early terminal displays. Today, it is primarily used for Unix permissions.
3. **How do I convert octal to binary?** Replace each octal digit with its equivalent 3-bit binary code (e.g., 5 = 101).
4. **Is it the same as a `chmod` command?** Yes, an octal number like `755` is the primary way the `chmod` command sets file permissions.

7. **Is 8 a valid octal number?** No. Octal only goes from 0 to 7. 8 in decimal is `10` in octal.
