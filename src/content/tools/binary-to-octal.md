### Frequently Asked Questions (FAQs)

1. **What is Octal?** A base-8 number system that uses digits 0 through 7.
2. **Why use Octal instead of Hex?** Historically, computer word sizes were multiples of 3. Today, it is primarily used for the 3-bit categories of Unix permissions.
3. **How do I convert binary to octal?** Group the binary digits into sets of three (from right to left) and translate each set into its octal digit.
4. **Is it the same as Base64?** No. Octal is base-8; Base64 is base-64. Octal is for server permissions; Base64 is for data transmission.
5. **What is a "Word"?** A group of bits that early computers used as their primary unit of processing (often 12, 18, or 36 bits).
6. **Can I use it for colors?** No. Colors use Base-16 (Hex). You strictly use Octal for server permissions and some low-level machine codes.
7. **Does "777" mean full access?** Yes. In Unix, 777 means every user can read, write, and execute the file.
